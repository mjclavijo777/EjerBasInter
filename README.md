Este proyecto es una simulación de un sistema de transporte desarrollado en Java, donde se aplican los principios fundamentales de la Programación Orientada a Objetos (POO).
El programa permite al usuario seleccionar un tipo de transporte (Taxi, Bus o Bicicleta), ingresar datos del recorrido (origen y destino) y visualizar la información del viaje, incluyendo el costo en algunos casos.
